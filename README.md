# marksheet
This project is a simple Java console-based Marksheet Generator that collects marks for 5 students, each with 7 units, and computes:

Total Marks

Average Marks

Displays a formatted marksheet table

It is suitable for beginners learning Java input handling, formatting output, loops, and basic calculations.
